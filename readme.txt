# javaScriptTipsTab

ENGLISH


ABOUT

Chrome extension to show random JS tips in new tabs! In English or Spanish.


HOW TO USE IT?

Simple.You can use it by accessing the website here: https://emicastro.github.io/javaScriptTipsTab/ or you can download the repository to your computer and use it like a chrome extension.


HOW TO MAKE IT A CHROME EXTENSION?

Open a new tab in Google Chrome and navigate to chrome://extensions. Inside this tab, click on the Developer Mode checkbox in the top right corner. Once that is checked, some buttons will appear. Click on Load Unpacked Extension…

This will open up a file browser. Navigate to your JavaScript Tips Tab project folder and select it. This enables the extension.

Now: Open a new tab!

ESPAÑOL



ACERCA DE

Extensión de Chrome para mostrar tips aleatorios de JavaScript cuando abrís una nueva pestaña! En inglés y en español.


¿CÓMO USARLA?

Sencillo. Podés utilizarla ingresando al sitio web https://emicastro.github.io/javaScriptTipsTab/ o descargar el proyecto desde este repositorio a tu computadora y utilizarlo como extensión de Google Chrome.

¿CÓMO LO VUELVO UNA EXTENSIÓN DE CHROME?

Abrí una nueva pestaña en Google Chrome y navega hasta chrome://extensiones. Dentro de esta pestaña, has click en el checkbox que dice Modo de Programador ubicado en la esquina superior derecha de tus extensiones. 

Una vez que esté marcado aparecerán una serie de botones. Hacé click en "Cargar extensión sin empaquetar..." Esto abrirá un buscador de archivos. Buscá la carpeta de JavaScript Tips Tab que descargaste del repositorio y selecciónala.

Ahora solamente abrí una nueva pestaña y ¡¡¡ BINGO !!! ya tenés la extensión funcionando.
